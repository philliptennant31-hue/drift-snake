## Adding a track

1. Confirm the license permits redistribution (CC BY, CC0, or the artist's
   written permission). Keep a note of where it came from.
2. Drop the audio file in this folder.
3. Add an entry to `tracks.js` (title, artist, link, license).
4. Add a row to this table.
